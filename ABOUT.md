# About Solarplan

## What is This Project?

Solarplan is a comprehensive technical guide for upgrading a boat's electrical system with renewable solar power and intelligent battery management. This project documents a complete system design for a **1.2kA marine solar installation** paired with dual-bank battery management, totaling 1200Ah of storage.

## Who Is This For?

This guide is designed for boat owners, marine electricians, and DIY enthusiasts who want to:
- Generate power sustainably using solar panels
- Implement robust battery management for extended cruising
- Install marine-grade electrical systems safely
- Follow industry standards for high-reliability marine installations
- Understand the Victron ecosystem for energy management

## What's Covered

The complete system includes:

### Hardware Components
- **Solar Arrays**: 1200W primary + 400W reserve generation capacity
- **Battery Banks**: 800Ah primary bank (Bank A) + 400Ah reserve bank (Bank B)
- **Power Electronics**: Victron Orion DC-DC charger, Renogy MPPT controllers, Magnum inverter
- **Monitoring**: Victron SmartShunt for real-time battery state-of-charge tracking
- **Protection**: Class T fuses, marine-grade cable, and proper grounding

### Documentation
- Complete bill of materials with sourcing recommendations
- Color-coded wiring schematics
- Step-by-step installation procedures
- Bank configuration and interconnection strategy
- Victron VictronConnect programming instructions
- Comprehensive commissioning and testing checklists

## Project Status

This project is actively maintained and documents proven installations. It follows marine-grade standards including UL 1426 for all bilge-transit cabling and professional marine electrical practices.
